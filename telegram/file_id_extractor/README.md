# Telegram File ID Extractor

A simple Telegram bot that extracts File IDs from Telegram media.

## Features

- Photo File ID
- Video File ID
- Animation File ID
- Audio File ID
- Voice File ID
- Document File ID

## Installation

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
BOT_TOKEN=YOUR_BOT_TOKEN
```

Run:

```bash
python bot.py
```
