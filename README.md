# Dev Toolkit

A curated collection of developer tools, Python utilities, Telegram bot examples, Docker snippets, automation scripts, and prompt engineering resources.

## Repository Structure

```
python/       Python utilities and helpers
telegram/     Telegram Bot API examples
docker/       Docker and Docker Compose examples
sqlite/       SQLite helpers and examples
automation/   Automation scripts
prompts/      AI prompts and prompt engineering
examples/     Miscellaneous code examples
```

## Goals

- Share reusable development utilities
- Keep code clean and well documented
- Publish practical examples for developers
- Build an open-source knowledge base
